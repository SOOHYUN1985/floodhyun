## ★ 베스트 시나리오 — 결론부터

> **담보 500%이므로 강제매도 위험이 없다. 방향을 모를 때 최선은 "오늘 아무것도 안 하고, 내일 갭을 보고 움직이는 것"이다.**

### 추천 행동: 옵션 A (현상유지) + 내일 갭 조건부 실행

```
오늘 (4/6): 아무것도 하지 않는다.
            → 3억 선매도는 상승 시 최대 0.3억 손해만 줄 뿐
            → 방향 모르는 상태에서 포지션 축소는 손해

내일 (4/7) 장 시작 후:

  ① 갭업 +5% 이상 (전쟁 종결 뉴스)
     09:30~10:00  3억 분할 매도 (1.5억 × 2회, 지정가)
     → 대출 3억 즉시 상환 → 대출 3억으로 축소
     → 순자산 ~25.5억 일부 확정, 잔여 27억은 계속 상승 향유
     → 추가 +10% 시: 오후 13:00~14:00에 3억 더 매도 → 대출 완전 상환

  ② 갭다운 -5% 이상 (전쟁 확장 뉴스)
     09:05~14:00  오전 관망. 하락일엔 오후가 가장 싸다
     15:10~15:20  여유자금 범위 내 추가 매수 검토
     → 담보 500% → -30% 폭락에도 담보 350%, 절대 공황매도 금지

  ③ 소폭 변동 ±3% 이하 (결정 연기 / 중립 내용)
     → 현상유지. 변동성 잦아들 때까지 그냥 보유
```

### 이 전략이 최선인 이유

| 비교 | 오늘 선매도(B) | 오늘 현상유지+내일 행동(추천) |
|:---|:---:|:---:|
| 갭업 +10% 시 | 0.3억 손해 | **+3억 풀 향유** ✅ |
| 갭다운 -10% 시 | 0.3억 유리 | **-3억 (버틸 여유 충분)** |
| 방향 결정 전 리스크 | 이미 포지션 축소 | 옵션 열어둠 ✅ |
| 실행 복잡도 | 오늘+내일 2번 행동 | 내일 1번만 행동 ✅ |

> 핵심: B(선매도)와 A(현상유지)의 차이는 **±0.3억**에 불과하다.  
> 그 차이를 위해 오늘 포지션을 줄이는 것보다,  
> **내일 갭업이 확인되면 그때 팔고 (D), 갭다운이면 더 사는 것 (C)** 이 훨씬 합리적이다.

## 3. 선택 가능한 4가지 옵션

### 옵션 A: 현상유지 (아무것도 안 한다)

**적합 조건**: 시장 방향에 대한 확신이 없을 때, 포지션에 만족할 때

```
장점                              단점
────────────────────────────────────────────────────
급등 시 수익 100% 향유           급락 시 손실 방어 없음
별도 실행 불필요                  심리적 불안 가능성
담보비율 500%로 충분히 안전      기회비용 없음
```

**내일 시나리오:**
- 급등 (+10%): 순자산 **27억** (+3억)
- 급락 (-10%): 순자산 **21억** (-3억)
- 결정 불발/지연: 현상유지

---

### 옵션 B: 일부 선매도 헤지 (3억 매도 + 대출 상환)

**개념**: 오늘 장 마감 전에 3억어치 매도 → 대출 3억 상환 → 예수금 0, 대출 3억 상태로 내일 진입  
→ 급락 시 대출 부담 줄고, 급락폭이 클수록 재매수로 **더 많은 수익** 

```
선매도 후 상태:
  평가 27억 / 대출 3억 / 순자산 24억 (동일) / 담보 900%
  예수금 있음 (재매수 탄약)
```

**내일 시나리오별 순자산 비교 (재매수 없이 보유만 할 때):**

| 변동 | 옵션A 순자산 | 옵션B 순자산 | 차이 (B−A) |
|:---:|:---:|:---:|:---:|
| **+10%** | 27.0억 | 26.7억 | **−0.3억** (B 불리) |
| **+5%** | 25.5억 | 25.35억 | **−0.15억** (B 불리) |
| **0%** | 24.0억 | 24.0억 | **동일** |
| **−5%** | 22.5억 | 22.65억 | **+0.15억** (B 유리) ✅ |
| **−10%** | 21.0억 | 21.3억 | **+0.3억** (B 유리) ✅ |

> 재매수 없이 보유만 할 경우: 상승 시 최대 **−0.3억**, 하락 시 최대 **+0.3억** 차이.  
> 이유: 같은 24억 순자산을 대출 3억(B) vs 대출 6억(A)으로 운용→레버리지 차이만 존재.  
> B의 진짜 이점은 급락 시 **재매수를 통해 더 많은 주식을 확보**하는 것 (아래 참조).

**급락 후 재매수 시나리오 (옵션B의 진짜 힘):**

```
예시: 내일 -10% 급락 → 15:20에 3억어치 재매수

  현상유지:        평가 27억, 대출 6억, 순자산 21억
  선매도+재매수:   평가 27억×(27/27)+재매수분, 대출 3억, 순자산 ?

  구체적으로:
  - 오늘 3억(27억의 11.1%) 매도, 대출 3억 상환
  - 내일 -10%에 3억어치 재매수 (이미 싸진 가격에 더 많이 삼)
  
  27억 × 0.9 = 24.3억 (보유분 평가)
  재매수 3억 추가 → 총 평가 27.3억
  대출: 기존 3억(남은것) + 신규 3억(재매수용) = 총 6억
  순자산 = 27.3 − 6 = 21.3억 vs 현상유지 21.0억
  → 선매도+재매수가 0.3억 유리

  수익이 작아 보이지만:
  - 재매수분을 10% 싸게 매입 → 주식 수량이 A보다 많아짐
  - 시장 완전 회복 시: B 순자산 24.33억 vs A 24.0억 (0.33억 영구 우위)
  - B의 진짜 가치: 숫자보다 "레버리지 축소 + 급락 시 냉정한 추가매수 실행력"
```

**옵션B 실행 방법:**

```
오늘 (4/6) 장 마감 20분 전:
  ① 15:10에 1.5억 매도 (시장가 아닌 지정가, 현재가-0.3%)
  ② 15:20에 1.5억 매도
  ③ 즉시 증권사 앱에서 3억 대출 상환

내일 (4/7) 행동 기준:
  ─ 갭업 +3% 이상 → 재매수 없이 보유 (상승 추세 확인)
  ─ 갭다운 -5% 이상 → 15:10~15:20에 3억 재매수 (더 싸게 회수)
  ─ 소폭 변동 ±2% → 장 상황 보고 판단
```

## 6. 결론: 어떤 옵션이 최선인가

| 상황 | 추천 옵션 | 이유 |
|:---|:---:|:---|
| 심리적으로 편안하다 | **A** | 담보 500%, 아무것도 안 해도 됨 |
| 약간 불안하다 | **B** | 3억 미리 팔고 대출 줄여두면 급락에도 편안 |
| 수익 일부 확보하고 싶다 | **D** | 갭업 순간 분할 차익 실현 |
| 적극적 승부 | **C** | 내일 갭 방향 보고 행동 |

**한 줄 결론:**  
담보 500%면 아무것도 안 해도 안전하다. 옵션B(선매도+재매수)의 실질 수익 차이는 **±0.3억 수준**으로 작다. B의 진짜 가치는 숫자가 아니라 심리적 여유와 "급락 시 냉정하게 사는 실행력"이다. 급등 시엔 재매수 안 하면 A 대비 0.3억 불리. **방향 확신이 없다면 A가 최선. 불안하면 B. 수익 일부 확정하고 싶으면 D.**
